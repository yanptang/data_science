目前的框架体系可以根据深度学习、传统机器学习以及大模型 (LLM) 应用的主要三个维度来划分
| 分类                | 框架/库名称                  | 核心优势与特点                                                                 | 典型应用场景                                   |
|---------------------|------------------------------|------------------------------------------------------------------------------|----------------------------------------------|
| 深度学习核心        | PyTorch                      | 动态图机制，生态极其丰富（Hugging Face 等），开发体验极佳。                   | 学术论文实现、LLM 微调、快速原型开发。         |
|                     | TensorFlow                   | 部署生态完善，TensorBoard 监控强大，静态图优化适合超大规模工业生产。           | 企业级大规模流水线、移动端及嵌入式部署。       |
|                     | JAX                          | 基于 XLA 编译，极速的线性代数运算，支持自动微分和硬件加速。                   | 高性能科学计算、复杂强化学习、底层算法创新。   |
| 传统机器学习        | Scikit-learn                 | 接口统一，包含了几乎所有经典的统计学和机器学习算法。                         | 基础算法教学、小规模数据集建模、特征工程。     |
|                     | XGBoost / CatBoost           | 梯度提升树（GBDT）代表，处理表格化（Tabular）数据性能顶尖。                   | 金融风控、点击率预测、Kaggle 竞赛。            |
| 大模型 (LLM) 生态   | Transformers                 | Hugging Face 出品，预训练模型的事实标准，一行代码调用各种开源 SOTA 模型。      | 文本生成、情感分析、多模态任务。               |
|                     | LangChain / LangGraph        | 构建 AI Agent（智能体）的胶水框架，支持复杂的逻辑链条和记忆。                 | 智能对话机器人、自动化 AI 工作流。             |
|                     | vLLM / TensorRT-LLM          | 推理性能怪兽，通过算子优化和显存管理极大提升模型响应速度。                     | 大模型 API 服务部署、私有化模型推理。          |
| 数据预处理          | Pandas / Polars              | Polars 凭借 Rust 底层性能在 2026 年成为处理千万级数据的首选。                  | 数据清洗、探索性数据分析 (EDA)。               |
| 向量检索            | Milvus / Pinecone            | 专门存储海量 embedding 向量，支持高并发的相似度检索。                         | RAG（检索增强生成）系统、知识库问答。           |
