# 1. 计算机基础技能

## 1.1 Python 基础与进阶

* 基础语法（变量、循环、函数、类）
* 数据结构（list、dict、tuple、set）
* 模块系统、虚拟环境
* 文件操作与数据读写
* 代码规范（PEP8）与常用工具

### Python 在数据科学中的必备库

* Numpy（矩阵运算）
* Pandas（数据处理、ETL）
* Matplotlib / Seaborn（可视化）
* Scikit-learn（机器学习基础）
* Jupyter Notebook（项目展示与实验）

---

## 1.2 SQL 与数据库

* 数据库基础概念（表、行列、索引）
* 基础查询（SELECT、WHERE、ORDER BY、LIMIT）
* 多表联结（JOIN）
* 分组与聚合（GROUP BY、HAVING）
* 子查询、窗口函数
* 数据建模与多源数据整合
* 大数据 SQL（Hive / BigQuery 简介）

---

## 1.3 数据工程基础（了解即可）

* 批处理 vs 流处理
* 文件格式（CSV / Parquet / ORC）
* 分布式计算（Spark 基础）
* 版本控制（Git）

# 2. 数学基础

## 2.1 线性代数（机器学习最关键）

* 向量、矩阵及运算
* 特征值、特征向量
* 矩阵分解（SVD、PCA 基础）
* 二次型与优化几何直觉

---

## 2.2 微积分

* 函数、极限、连续性
* 导数、偏导数、梯度
* 多元函数与梯度下降
* 链式法则（深度学习核心）

---

## 2.3 概率论与统计

* 随机变量与分布（高斯、伯努利、均匀…）
* 数学期望、方差、协方差
* 概率密度函数、条件概率
* 贝叶斯公式 ⭐
* 大数定律、中心极限定理
* 假设检验（t-test、p-value）⭐
* 置信区间
* A/B 测试原理⭐
* 因果推断⭐

# 3. 商业沟通与可视化能力⭐

## 3.1 数据可视化

* 图表原则（避免误导与过度渲染）
* 常用图表类型（折线、柱状、箱线、热力图）
* 可视化工具（Matplotlib、Seaborn、Tableau、Power BI）
* 如何用图讲故事（Data Storytelling）

---

## 3.2 报告与文档写作

* 如何撰写清晰的数据分析报告
* 结构化表达（背景 → 方法 → 结果 → 建议）
* 面向技术与业务方的不同写法
* 图文排版与结果展示规范

---

## 3.3 业务沟通与汇报技巧

* 如何理解业务问题
* 如何将数据转换成业务语言
* 如何用数据驱动决策（Data-Driven Decision Making）
* 如何回答“这意味着什么？”
* 如何做电梯演讲（结论先行）

# 4. 数据预处理与特征工程

## 4.1 数据清洗

* 缺失值、异常值、重复值处理
* 离群点检测方法
* 数据一致性检查

---

## 4.2 特征工程

* 数值特征处理（归一化、标准化）
* 类别特征处理（One-Hot、Embedding）
* 时间特征提取
* 文本特征（TF-IDF、词袋模型）
* 特征选择（过滤法、包装法、嵌入法）
* 降维技术（PCA、t-SNE）

# 5. 机器学习模型（核心章节）

## 5.1 监督学习

* 线性回归
* 逻辑回归
* 决策树、随机森林、XGBoost
* SVM（支持向量机）
* KNN
* 神经网络（MLP）

---

## 5.2 无监督学习

* 聚类（K-means、DBSCAN）
* 降维（PCA、ICA）
* 异常检测

---

## 5.3 模型评估⭐

* 数据划分（Train/Val/Test）
* 分类指标（Accuracy、Precision、Recall、F1）
* ROC、AUC
* 回归指标（RMSE、MAE）
* 交叉验证
* 避免过拟合的方式
